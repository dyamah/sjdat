概要

静的なダブル配列によるTrieのJava実装です。これまで原理は（なんとなく）わかっていたダブル配列たが、勉強もかねて実装

動作環境

Java実行環境(JRE 1.7 以降)

ビルド方法

ビルドには maven 3.x以降が必要。

githubリポジトリから最新のソースコードをチェックアウト。 プロジェクトのルート・ディレクトリで以下のコマンドを実行

% mvn package

target/ ディレクトリ以下に sjdat-1.0-SNAPSHOT.jar が作成される。

使用方法

(1) [ダブル配列構築時]

% java -cp sjdat-1.0-SNAPSHOT.jar com.github.dyamah.sjdat.tools.CLT -b file1 file2

file1: オプションに続き、Trie をセーブする保存先ファイル

file2: ダブル配列に登録するキーリストのファイル。このファイルは一行に１キーで改行しているテキストデータ。 未ソート、重複、空行があるとダブル配列構築時に例外がスローされる。

(2) [検索時]

% java -cp sjdat-1.0-SNAPSHOT.jar com.github.dyamah.sjdat.tools.CLT -s file1 file2

file1: (1)で構築したダブル配列のファイル。

file2: 検索するキーリスト。書式は(1)と同じ。出力は標準出力
